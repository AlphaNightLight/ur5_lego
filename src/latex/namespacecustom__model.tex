The function that make effective prediction given the image based on the yolo v8 fine-\/tuned model create. 

This function computes size of given image and then uses yolo v8 model.\+predict pre-\/defined function that run inference on the source, to which I pass following values defined by real world experience\+:
\begin{DoxyItemize}
\item source = path -\/-\/$>$ takes that specific video of image path.
\item source = 1 -\/-\/$>$ use the webcam of the pc, 0 to the external camera
\item show = True -\/-\/$>$ shows webcam view
\item conf = 0.\+3 -\/-\/$>$ object confidence threshold for detection
\item save = True -\/-\/$>$ save predicted images and videos)
\item iou\+\_\+thres = 0.\+7 -\/-\/$>$ intersection over union threshold value Set to True or false variable A\+B\+S\+O\+L\+U\+T\+E\+\_\+\+V\+A\+L\+UE if you want to have relative or absolute measurement
\end{DoxyItemize}

\begin{DoxyItemize}
\item ID -\/-\/$>$ from 0 to 10
\item prec -\/-\/$>$ from 0 to 100 (it is a percentage)
\item xc -\/-\/$>$ from 0 to x size of image (x coordinate of center of bounding box with absolute coordinate)
\item yc -\/-\/$>$ from 0 to y size of image (y coordinate of center of bounding box with absolute coordinate)
\item w -\/-\/$>$ from 0 to x size of bounding box (width of bounding box with absolute coordinate)
\item h -\/-\/$>$ from 0 to y size of bounding box (height of bounding box with absolute coordinate)
