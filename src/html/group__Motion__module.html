◆ differential_inverse_kin_quaternions()

Path differential_inverse_kin_quaternions ( Vector8d  mr,
Vector3d  f_p,
Quaterniond  f_q 
)

To compute the path the robot has to follow to reach the final point, which is expressed using position and quaternion for orientation.

The computation is performed by firstly adding the starting position to the path and then the following ones are added with an increment of dt=0.1s. Joint velocities are computed using the Damped Least Square (DLS) Inverse, to avoid problems in inverting differential kinematics near a singularities.

Parameters
[in]mrare the theta angles of joints plus 2 variables to express gripper state.
[in]f_pfirst quaternion.
[in]q2second quaternion.
Returns
a matrix with 8 columns and a variable number of rows.

qerr_k = Kq * qerr_k.coeffs();

Quaterniond total_q; total_q = qv_k * qerr_k;

To insert a new path instance each time it's computed using inverse differential kinematics.

Path is a matrix with dynamic-rows-number, hence each time a path instance (a new set of joint variables) is compute in inverse differential, it's added to path.

Parameters
[in]pis the path the robot has to follow. It consists in a matrix with 8 columns (6 for the joints + 2 for the gripper) and a dynamic number of rows.
[in]jsjoint variables.
[in]gsgripper state.
Returns
a matrix with 8 columns and a variable number of rows.
