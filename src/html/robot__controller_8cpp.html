Detailed Description

Ros node that exposes a move_robot service of type ur5_lego::MoveRobot. It recieves a vector of 8 joints, and moves the robot to that configuration from whatever pose it is now sending a message on the /ur5/joint_group_pos_controller/command topic of ur5_generic.py.

handler of move_robot service, type ur5_lego::MoveRobot.

It retrieves the actual joint state of the robot and interpolates it with the desired one. All the intermediate joints are sent to the robot, causing the motion.
